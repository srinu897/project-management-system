# Project Management System

A full-stack Project Management System built using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript.

## Features

* User Authentication (Signup/Login)
* Project Dashboard
* Task Management (Kanban Board)
* Document Upload & Management
* Search and Filtering
* Responsive SaaS UI
* MySQL Database Integration

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MySQL
* File Uploads: Multer

## Installation

```bash
npm install
node ss.js
```

Server runs at:

```text
http://localhost:8089
```
